WTPESTORE — App Icons (Option E: Drop + WTPeSTORE naam)
========================================================

UPLOAD KAHAN:
1. icons/ folder ki 16 files  →  GitHub repo ke "icons" folder me (purani replace kar do)
2. apple-touch-icon.png        →  repo ki ROOT me (purani replace)
3. favicon.ico                 →  repo ki ROOT me (purani replace)

BAAKI KUCH NAHI BADALNA — manifest.json aur HTML pehle se sahi hain.

Upload ke baad:
- 5 minute ruko
- Phone me app pehle se install ho to usse HATA do (uninstall), phir dobara install karo
  (Android purana icon yaad rakhta hai)
- Ctrl+Shift+R se site refresh karo

NOTE: Browser tab wala chhota icon (16/32px) me sirf drop rakha hai —
kyunki tab itna chhota hota hai ki naam wahan dhabba lagta. App icon me poora logo hai.
