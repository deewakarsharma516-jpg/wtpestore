SEO Page Generator

🔍 Google ke liye product pages banayein

Yeh tool aapki Google Sheet padh kar products.html aur sitemap.xml banata hai — jisme har product ka naam, model, brand, spec aur price asli text me hota hai, taaki Google use padh sake.

Files banne ke baad:
  1. Dono file download karein (buttons upar aayenge)
  2. GitHub repo → Add file → Upload files → dono drop karein → Commit
  3. Search Console → Sitemaps → sitemap.xml submit
Roz apne aap update chahiye? Zip me di gayi build-seo.js + .github/workflows/seo-update.yml upload kar dein — phir yeh kaam GitHub roz khud kar lega, aapko yahan aana bhi nahi padega.
